    Action Committee
    for
    Women in Prison

    The ACWIP advocates for the humane and compassionate treatment of incarcerated women; specifically focusing on incarcerated women with children.

    The ACWIP’s original website is very cluttered, confusing and difficult to navigate through. With their website being in that state, they run the risk of someone who is looking for more information or worse yet, someone who wants to get involved, clicking away from ACWIP’s site to find one that they can more easily interact with.

    In order to clearly communicate with the user and capture their attention I designed and engineered the following:

    • A clean homepage with quick links to information about the organization, their work and how to become a volunteer
    • Linked pages with clear and easily digestible content
    • A built-out volunteer portal to allow people to quickly register to become a volunteer and immediately get involved with the organization

    Category: Full Stack CRUD Application

    Technolgies used: React, JavaScript, HTML, Redux Sagas, PostgreSQL, CSS and Bootstrap

    MyMovies

    MyMovies is a React application that allows users to search for movies and manage their watched movie list. The app retrieves movie data from the OMDB API and provides features to search for movies, view movie details, and track watched movies along with user ratings.

    Features

    • Users can search for movies by entering keywords.
    • Displays a list of movies based on the search query.
    • View detailed information about a selected movie.
    • Tracks movies that the user has watched along with user ratings.
    • Provides an intuitive user interface with buttons for adding movies to the watched list, rating movies, and deleting movies from the watched list.
    • Designed for seamless viewing across devices

    • Technologies used:

      HTML5, CSS, JAVASCRIPT and REACT

    PACKD

    PackD is a packing list app built using React that helps you keep track of items you need to pack for your trip. You can add items, mark them as packed, delete items, and clear the entire packing list.

    Features

    • Add items to your packing list along with their quantity
    • Mark items as packed with a checkbox
    • Delete individual items from the list
    • Sort items by input order, description, or packed status
    • View statistics on the percentage of items packe

    Designed for seamless viewing across devices

    • Technologies:

      HTML5, SCSS, JS and React
